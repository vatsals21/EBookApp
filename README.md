# EBookApp
This app allows user to read, buy books online

## Introduction
The aim of the project is to allow the users to read or buy any available book for a nominal fees. The project will use React Native framework, JavaScript.

## Notes
1. To change app name, go to android>app>src>main>res>values>strings.xml and change the name under resources tag.
2. To change the app logos, go to android>app>src>main>res. Change all the images inside the below mentioned folders -
  a. mipmap-hdpi - dimensions - 72*72
  b. mipmap-mdpi - dimensions - 48*48
  c. mipmap-xhdpi - dimensions - 96*96
  d. mipmap-xxhdpi - dimensions - 144*144
  e. mipmap-xxxhdpi - dimensions - 192*192
  Remember to add rounded image as well.
